Data Security Practices

We safeguard data used by Tribal Goods Trail through the following controls:

Platform and Storage

  • Data is stored in Firebase with encryption at rest and in transit (TLS).
  • Back-end access is limited to the verified administrator via authenticated consoles.
  • Access is logged through provider tooling where available.

Data Minimization

  • We collect only the submission details needed to list businesses and support verification.
  • We do not store precise location history; proximity is calculated on the device when possible.
  • We avoid collecting sensitive personal data that is not required for the service.

Sharing and Third Parties

  • No sale of personal data.
  • Limited sharing with trusted service providers (e.g., Firebase) to operate the app.
  • No third-party marketing trackers are provided with personal information.
